# Evaluation Git - TSSR 25-06

##  Project Overview

This repository contains the work completed as part of the Git evaluation for the TSSR training program.

The objective of this project is to demonstrate the ability to:

* Use Git commands in a local environment
* Work with a remote repository (GitHub)
* Manage branches and workflows
* Perform commits, push, pull, and pull requests

##  Git Workflow

The project follows a simple Git workflow:

* `main` branch: stable version of the project
* `feature-loic` branch: development branch for new features

Steps performed:

1. Creation of the main structure on `main`
2. Creation of a feature branch (`feature-loic`)
3. Development of MVC structure and login view
4. Modification of files on both branches
5. Creation of a Pull Request
6. Merge of the feature branch into `main`
7. Conservation of the feature branch after merge
